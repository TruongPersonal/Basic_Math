# Basic Math Form 📜

## Overview
Basic Math is a simple Java Swing application that allows users to perform basic arithmetic operations: addition, subtraction, multiplication, and division. The application provides a user-friendly interface with input validation and real-time error messages.

## Features
- Perform **addition, subtraction, multiplication, and division**.
- Input validation with error messages for empty or invalid inputs.
- Handles **integer and floating-point numbers** correctly.
- Displays results dynamically based on input precision.

## Prerequisites
- **Java Development Kit (JDK) 8 or higher**
- **Eclipse IDE** (or any compatible Java IDE)

## How to Run
1. **Clone or download** the project files.
2. **Open Eclipse IDE**.
3. **Import the project**:
   - Select **File > Import > Existing Projects into Workspace**.
   - Browse to the project directory and click **Finish**.
4. **Run the application**:
   - Open `BasicMathForm.java`.
   - Click **Run** or press `Ctrl + F11` on Windows or `Command + F11` on MacOS to execute the program.

## Usage
1. Enter **two numbers** in the provided input fields.
2. Click the corresponding operation button (**Sum, Difference, Product, Quotient**).
3. The **result** will be displayed in the result field.
4. If input is invalid, an **error message** will appear next to the respective field.

## Notes
- **Division by zero** is handled with a proper error message.
- The result format dynamically adjusts based on input precision.
- Works with both **integers and floating-point numbers**.
